# SKILL_Image_Generator

이 저장소는 `@a1-image-generator`로 불러서 사용할 수 있는 Codex 스킬입니다.

## 사용 방법

1. 먼저 참고 이미지나 텍스트가 있는지 묻습니다.
2. 있으면 이미지를 올리거나 텍스트를 붙여넣습니다.
3. 총 몇 장을 만들지 묻습니다.
4. 어디에 저장할지 묻습니다.
5. 원하는 비율이 있는지 묻습니다. 없으면 기본값 `1:1`을 사용합니다.
6. 그다음 어떤 방향으로 만들지 묻습니다.
7. 생성된 이미지는 기본적으로 `C:\Users\tester\.codex\generated_images\`에 저장됩니다.

## 포함 파일
