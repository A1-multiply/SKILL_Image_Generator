# SKILL_Image_Generator

이 저장소는 `@a1-image-generator`로 불러서 사용할 수 있는 Codex 스킬입니다.

## 사용 방법

1. 스킬을 실행하면 먼저 참고 이미지나 텍스트가 있는지 묻습니다.
2. 있으면 이미지를 올리거나 텍스트를 붙여넣습니다.
3. 방향을 정합니다. 예: 실사, 광고, 썸네일, 제품컷, 유사 스타일.
4. 총 몇 장을 만들지 입력합니다.
5. 생성된 이미지는 기본적으로 `a1-image-generator/image/`에 저장됩니다.

## 포함 파일

- `a1-image-generator/SKILL.md`
- `a1-image-generator/agents/openai.yaml`
- `a1-image-generator/image/.gitkeep`
